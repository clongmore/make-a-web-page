SIMPLIFYING CONCEPTS

LESSON 1: The Basics of the Web and HTML

INTERNET

The Internet is a structure made up of millions of computers whose users can communicate with each other and share information.

WORLD WIDE WEB (WWW)

The WWW gives Internet users easy access to information on the Internet.

HTML

HTML is Hypertext Markup Language and it is the main, or most common, type of document on the web. There are other types such as text, images, videos, and music. HTML was designed to describe the contents of a web page in a very general way.

HTTP

HTTP is Hypertext Transfer Protocol that sends requests to a server.

Computers are stupid because they can only do what programmers tell them to do.

CODING

Coding is programmers writing in a language that the computer can read.

Let's take a look at some of the basic HTML vocabulary.

TAG: Turns on a feature which is indicated by an opening tag (< >), and turns off a feature indicated by a closing tag (< />).

ELEMENT: Everything within a set of opening and closing tags.

INLINE ELEMENTS: Tags that do not require a closing tag.

BLOCK ELEMENTS: Tags that require a closing tag. Block Elements are similar to an invisible box being placed around the text.

ATTRIBUTE: A property of an HTML element.

<B> There is a tag to do BOLD text.

<EM> There is a tag to do emphasis such as italics.

<BR> There are tags to do line breaks.

<P> There are also tags to do paragraphs.

There are many others which I look forward to learning in this class. There are tags to do almost everything. After all, computers are stupid and will only do what we tell them to do, so we have to talk to them in a language they can understand--HTML

LESSON 2: Creating Structured Documents with HTML

The structure of a web page can be determined by looking at its coding or markup language (usually HTML or CSS).

HTML (Hypertext Markup Language)

HTML is a programming language that controls the structure of a web page (how it is set up).

CSS (Cascading Style Sheet)

CSS is a programming language that controls the style of a page (how it looks).

DOM (Document Object Model)

DOM is the tree-like structure of a page. You can relate the tree-like structure to a family tree. The tree trunk is the main page. The limbs are pages connected to the trunk. The branches are pages that are connected to the limbs.

DIV Tag <div>

The <div> tag defines boxes. Give each <div> tag a class element to better define the tag.
